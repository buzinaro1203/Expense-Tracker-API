# Documento do escopo do projeto
## Definição do problema
O controle financeiro é uma atividade cotidiana comum. Porém para muitas pessoas pode ser uma atividade complexa e difícil de acompanhar para onde seu dinheiro está indo.

Sem um registro estruturado dessas informações o acompanhamento fica ainda mais complicado. Principalmente ao tentar entender padrões de consumo, identificar gastos excessivos ou analisar seu perfil financeiro.

Dessa forma, surge a necessidade de uma ferramenta que ajude a registrar e acompanhar esses dados de forma simples para que o usuário consiga analisar e entender seu consumo.

## Usuário do sistema

O sistema é destinado a pessoas que desejam acompanhar e organizar suas próprias despesas pessoais.

Cada usuário utiliza o sistema para registrar e consultar seus gastos ao longo do tempo, com o objetivo de ter maior controle sobre suas finanças e compreender melhor seus hábitos de consumo.

Os dados registrados no sistema pertencem exclusivamente ao usuário que os criou. Não existe compartilhamento de informações entre diferentes usuários.

Para o escopo inicial do projeto, o sistema considera apenas um tipo de usuário, responsável por registrar, consultar e organizar suas próprias despesas.

### Escopo funcional inicial

#### Gerenciamento de despesas
- Registrar despesas
- Editar despesas
- Remover despesas
- Listar despesas

#### Gerenciamento de rendas
- Registrar rendas
- Editar rendas
- Remover rendas
- Listar rendas

#### Gerenciamento de categorias
- Criar categorias
- Editar categorias
- Remover categorias

#### Consultas e relatórios
- Consultar gastos por período
- Consultar gastos por categoria
- Consultar renda bruta
- Consultar renda líquida

## Fora do escopo

Para a versão inicial do projeto, algumas funcionalidades comuns em sistemas financeiros não serão consideradas. Entre elas:

- Interface gráfica para interação com o sistema.
- Suporte a múltiplas moedas.
- Validação avançada de email.
- Integração com serviços bancários.
- Importação automática de extratos financeiros.
- Compartilhamento de contas entre diferentes usuários.
- Funcionalidades avançadas de planejamento financeiro.
